### lx-radio API
```javascript
{
  'key': 'modelName',
  'type': 'lx-radio-buttons'
  'templateOptions': {
    'label': 'Choose an item: ',
    'options': [{
      'name': 'item 1',
      'value': '1', // ng-value
      'help': 'item 1 help description',
      'checked': false, // ng-checked
      'disabled': false, // ng-disabled
    }]
  }
}
```
Read more about [LumX radio buttons](http://ui.lumapps.com/css/radio-buttons).
